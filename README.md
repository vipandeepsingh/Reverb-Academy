# Reverb Academy Website

Static website for Reverb Academy guitar and ukulele lessons in Mohali.

Open `index.html` directly in a browser, or deploy the folder as a static site on GitHub Pages, Netlify, or Vercel.

The site includes:

- Reverb Academy brand assets
- lesson feature cards
- teaching reel videos
- Google review cards
- map embed
- call and WhatsApp pricing buttons
